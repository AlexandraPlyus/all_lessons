# all_lessons - хранилище домашних работ по курсу Автоматизация тестирования на Python 

### Как запустить тесты для формирования отчета
1. Открыть папку ALL_LESSON, переместиться в терминале в папку 10_lesson (cd 10_lesson).
2. Запустить команду pytest --alluredir=./allure-result и дождаться окончания тестирования.
3. Запустить команду allure serve allure-result.
